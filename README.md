# Sales & Strategic Emails 🎯

**AI agent skills for sales emails, proposals, negotiation, and conflict resolution**

A curated collection of Hermes Agent skill files that encode proven sales methodologies, negotiation frameworks, and strategic philosophies. Each skill is a self-contained Hermes skill (`SKILL.md` with YAML frontmatter) that any AI agent can load and apply immediately.

## 🚀 Usage

### As a Hermes Agent skill

Any Hermes-powered agent can load a skill by name:

```bash
# Load a single skill
hermes skill load chris-voss-negotiation
```

Or in a conversation:
> "Load the Chris Voss negotiation skill and help me prepare for this client meeting"

### As a standalone reference

Each `SKILL.md` is self-contained with:
- Frontmatter metadata (name, description, toolsets)
- Core principles and frameworks
- Step-by-step application guides
- Real email templates and scripts
- Pitfalls and common mistakes

---

## 🧠 Philosophy

These skills are designed for **agentic use** — an AI agent should be able to:
1. Load a skill based on the sales situation
2. Follow its methodology to draft emails/proposals
3. Apply the right tone, structure, and persuasion technique
4. Combine multiple skills when appropriate (e.g., Voss + Yin Yang for a rebuff)
